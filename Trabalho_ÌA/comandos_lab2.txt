Arquivo – comandos_lab2 

 

-Criar o ambiente virtual 

 python –m venv lab2 

 

-Ativar ambiente virtual 

 lab2|Scripts|activate 

 

-Instalar o arquivo requirements 

 pip install –r requirements.txt 

 

-Para desativar este comando só pode ser usado assim que terminar de desenvolver ou não precisar mais usar o ambiente de desenvolvimento 

 desactivate
